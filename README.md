# Classificador-de-Reclamacoes

Projeto da disciplina de Inteligência Artificial, a ideia é utilizar o algoritmo de KNN (K-Nearest Neighbors) para classificar reclamações de usuários

# Objetivo

Classificar a reclamação do usuario em 1 das 10 labels possiveis:

- ÁGUA, ENERGIA, GÁS
- ALIMENTOS
- EDUCAÇÃO
- HABITAÇÃO
- LOTERIAS, APOSTAS E PROMOÇÕES COMERCIAIS
- PRODUTOS DE TELEFONIA E INFORMÁTICA
- PRODUTOS ELETRODOMÉSTICOS E ELETRÔNICOS
- SAÚDE
- SERVIÇOS FINANCEIROS
- TELECOMUNICAÇÕES
- TRANSPORTES
- TURISMO/VIAGENS
